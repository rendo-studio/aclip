# Notes Overview

Use this skill when the task spans multiple notes commands or when you need to verify prerequisites before editing data.

Start with `note list` to inspect the current store, then use `note create` for additive writes.

If the caller is unsure where notes are stored, ask for or infer the `--store` path before writing.
