# Note Create Best Practice

Use this skill when a caller needs to create a new note reliably.

Confirm the target store path first if the workspace uses anything other than the default `.aclip-demo-notes.json`.

Prefer a short, explicit title and keep the body content fully formed before invoking the command so the write is atomic.
